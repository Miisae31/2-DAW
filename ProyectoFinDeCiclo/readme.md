# MovieLoot

**MovieLoot** es una plataforma web para amantes del cine y las series donde los usuarios pueden descubrir, valorar y comentar sobre películas y series. Además, los usuarios pueden participar en foros de discusión sobre sus títulos favoritos, compartir teorías y críticas, y acceder a contenido exclusivo mediante suscripciones premium.

## ¿Qué puedes hacer en MovieLoot?

- **Buscar y explorar películas y series** mediante un buscador avanzado con filtros.
- **Dejar valoraciones y críticas** para cada película o serie que visualices.
- **Participar en foros** dedicados a películas, donde puedes debatir, compartir opiniones y teorías.
- **Acceder a contenido exclusivo** mediante suscripción premium, incluyendo la API para integrar datos de películas en otras plataformas.

## Tecnologías Utilizadas

- **Frontend**: Angular
- **Backend**: Laravel
- **Base de datos**: MySQL
- **Autenticación**: Laravel Sanctum
- **API**: Integración con la API de IMDb

MovieLoot es una plataforma creada para facilitar la interacción entre los cinéfilos y mejorar la forma en que se comparte y descubre contenido relacionado con películas y series.
